# Microgrid Code

Bachelor project repository.

## Setup

```bash
python -m venv bsc-proj-venv
source bsc-proj-venv/bin/activate
pip install -r requirements.txt

```
## Data

Large CSV files are not stored in this repository.

All CSV datasets must be placed inside:

microgrid-code/data/

Example:

microgrid-code/data/prices.csv
microgrid-code/data/load_profiles.csv
